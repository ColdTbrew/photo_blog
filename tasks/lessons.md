# Lessons

## 2026-02-22

- Pattern:
  - 사용자가 제외 조건을 명시하면(예: GPU 설정 변경 금지), 적용 전후에 제외 항목을 체크리스트로 고정한다.
- Rule:
  - 설정 변경 요청은 `추가/변경할 항목`과 `건드리지 말아야 할 항목`을 분리해 실행하고, 결과 보고에도 둘 다 명시한다.
- Pattern:
  - 사용자가 후속으로 범위를 확장 요청하면(예: 터미널 후 에디터 폰트 추가), 기존 반영 상태를 유지하면서 필요한 키만 최소 추가한다.
- Rule:
  - 설정 변경은 "기존 값 보존"을 기본 원칙으로 두고, 요청된 키만 증분으로 적용한다.
